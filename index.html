An Teasáras Leictreonach

Kevin P. Scannell

Summary

This is the home page for An Teasáras Leictreonach, a free Irish language thesaurus in hypertext PDF which we believe to be the first machine-readable monolingual thesaurus in a minority language.

Features

  • There are nearly 75,000 entries representing 40,000 unique words or phrases, organized into approximately 1000 main headings, similar to Roget's English language thesaurus.
  • For maximum portability across computing platforms and to ensure high quality printed versions, the thesaurus will be distributed in Portable Document Format (PDF).
  • Hypertext cross references can be used to navigate the thesaurus using the free Adobe Acrobat Reader.
  • Bibliographical sources are provided for each word.
  • The thesaurus is released under the GNU Free Documentation License. This means, in short, that you have the freedom to copy and redistribute the thesaurus, with or without modifying it, as long as you do so under the same license.
  • Full LaTeX source is available, making customization very easy. For instance, you can typeset using seanchló by downloading the appropriate fonts.
  • We expect to provide frequent updates, incorporating corrections and refinements, but also reflecting Irish as a living language via new terminology, shifting usages, etc.
  • The database used to generate the thesaurus is the same one we use to generate the GaelSpell family of spellcheckers. Improvements to one product will be reflected in the other automatically.

Contributing

The thesaurus is the unedited output of a sequence of complicated computer algorithms which transforms the underlying lexicographical database into the final typeset form. This facilitates change since one need only update the underlying database and the change should manifest itself properly in the finished product. We have found, however, that the algorithms are far from perfect and substantial human assistance will be needed to refine the separation of senses within entries. Please write to me at scannell@slu.edu if you are interested in contributing.
